Catatan Belajar

Berikut hal-hal penting yang sudah dipelajari di Web Client Development:

  1. Struktur HTML setiap halaman web wajib punya DOCTYPE, html, head, dan body.
  2. Tag formatting teks seperti bold, italic, underline, highlight, maupun strong.
  3. Perbedaan <b> dan <strong> → keduanya tebal, tapi strong punya makna semantik.
  4. Cara membuat hyperlink ke google dengan tag <a href dan menyisipkan gambar dengan <img src>.
  5. Penggunaan inline style untuk font, color, dan spacing.

Topik yang masih perlu diperdalam:

  • CSS Selector terutama perbedaan class dan ID.
  • Box Model memahami margin, padding, dan border secara visual.
  • Konsep specificity aturan mana yang menang saat ada konflik style.

Refleksi Pengerjaan

Approach

Saya memulai tugas ini dengan menyusun struktur dasar HTML terlebih dahulu. Kemudian saya membuat tabel mata kuliah menggunakan tag table, tr, th, dan td, serta mengombinasikan rowspan untuk jadwal yang memiliki dua hari. Setelah itu, saya melengkapi bagian daftar menggunakan ordered dan unordered list, serta menambahkan hyperlink ke Google sesuai instruksi.

Catatan

Tantangan utama dalam tugas ini adalah menyusun baris tabel (rowspan) agar presisi dan sesuai dengan gambar contoh tanpa menggunakan CSS layouting tambahan karena adanya batasan aturan. Namun, tugas ini sangat membantu dalam memantapkan pemahaman dasar tag HTML murni.
